`ctrl k + ctrl up` -> mover tab a new panel
`ctrl k + ctrl num` -> fold level num
`ctrl k + ctrl space` -> set mark
`ctrl k + ctrl a`          -> select to mark
`ctrl k + ctrl c` -> centrar en vertical
